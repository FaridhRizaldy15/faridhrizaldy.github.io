## ğŸ¨ Customization

### Mengubah Informasi Personal

Edit file-file component di `src/components/` untuk mengubah:
- Personal info di `About.jsx`
- Skills list di `Skills.jsx`
- Projects di `Projects.jsx`
- Contact info di `Contact.jsx` dan `Footer.jsx`

### Mengubah Theme Colors

Edit CSS variables di `src/index.css`:

```css
:root {
  --primary-color: #0d6efd;
  --secondary-color: #e2edff;
  /* ... */
}
```
